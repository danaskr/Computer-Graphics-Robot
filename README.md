# Computer Graphics Project – Animated Robot

The application renders a 3D scene containing a textured table and an animated robot using C++ and OpenGL.

## Features

### Scene
- 3D table built from cube geometries
- Textured robot model loaded from OBJ files
- Hierarchical transformations for robot body parts

### Graphics
- Phong lighting model
- Textured objects
- Custom shader effects:
  - Checkerboard table pattern
  - Robot shininess texture

### Animation
- Robot arms and head controlled with GUI sliders
- Animated leg movement
- Robot rotates and moves toward selected points

### Interaction
- Mouse click selects target position
- Adjustable movement speed
- Third-person camera mode
- Debug rendering modes

## Technologies

- C++
- OpenGL
- GLSL Shaders
- ImGui

## Learning Goals

- 3D transformations
- Hierarchical modeling
- Shader programming
- Lighting models
- Interactive graphics
